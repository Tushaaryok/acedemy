📱 Mobile-First Rules

  • All tap targets minimum 44×44px
  • Bottom navigation — thumb-friendly zone
  • No hover states on mobile — use tap
  • Swipe gestures for tabs and carousels
  • Loading skeletons — never blank screens
  • Haptic feedback on button press (Android)

🎨 Visual Hierarchy Rules

  • Blue (#1E3A8A) = primary actions and headings
  • Orange (#EA580C) = urgency — deadlines, live
  • Green (#15803D) = success — done, submitted, correct
  • Red (#DC2626) = danger — missed, wrong, alert
  • Max 3 font sizes per screen — H1/Body/Caption
  • 1 primary CTA per screen — never compete

⚡ Micro-interactions

  • Button press → scale(0.97) + shadow reduce
  • Live badge → pulsing red dot animation
  • Progress bar → animated fill on page load
  • Streak badge → fire emoji bounce on login
  • OTP boxes → auto-advance on digit entry
  • Test answer select → border flash + haptic

📐 Spacing & Layout Rules

  • Page margin: 16px mobile, 24px tablet, 32px desktop
  • Card padding: 12px (compact), 16px (standard), 24px (hero)
  • Section gap: 24px mobile, 40px desktop
  • Bottom nav height: 56px (Android standard)
  • Input height: 44px (minimum touch target)
  • Modal max-height: 80vh with scroll inside
